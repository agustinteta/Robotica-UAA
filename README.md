## Descripción

Este repositorio contiene los proyectos y ejercicios realizados en las clases de la materia de Robótica, desarrollados con Arduino. Cada carpeta corresponde a un trabajo práctico o ejercicio de clase que explora sensores, entradas/salidas, movimiento y control.

## Contenido por carpeta

- `clase2_entradasysalidas/`
  - Ejercicio de entradas y salidas básicas con Arduino.
- `clase3_movimiento/`
  - Trabajo práctico sobre movimiento y control de actuadores.
- `clase3_movimiento_linea_recta/`
  - Ejercicio para realizar movimiento en línea recta.
- `clase5_correccion_integral/`
  - Implementación de un control integral para corrección en el sistema.
- `clase5_sensor_proximidad/`
  - Proyecto con sensor de proximidad para detección de objetos.
- `clase6_arduinoday/`
  - Ejercicio desarrollado durante Arduino Day o actividad especial.
- `clase7_movimiento_giro_con_sensor/`
  - Trabajo con movimiento y giro controlado por sensor.
- `clase7_sensor_temperatura/`
  - Ejercicio con sensor de temperatura.
- `clase8_movimientos_wifi/`
  - Proyecto de movimiento controlado vía Wi-Fi.

## Instrucciones de uso

1. Abrir la carpeta del proyecto correspondiente en el IDE de Arduino.
2. Cargar el archivo `.ino` en el IDE.
3. Seleccionar la placa Arduino y el puerto serie adecuados.
4. Compilar y cargar el sketch en la placa.
